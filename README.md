## Project Description

This project implements a lightweight multi-container runtime using C language.

It contains:

- User-space runtime (`engine.c`)
- Kernel-space memory monitor (`monitor.c`)
- Workload programs
- Logging system
- Supervisor process

---

## Build Commands

```bash
make
sudo insmod boilerplate/monitor.ko
```markdown
